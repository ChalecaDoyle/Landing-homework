Summary:Latitude vs. X

The purpose of this project was to analyze to how weather changes as you get closer to to the equator. To accomplish this analysis, we first pulled data from the Open WeatherMap API to assesemble a datasheet on over 500 cities.

After assemble the dataset , we used Mateplotlib to plot various aspects of the weather vs. lattitude. Factors we looked at include: temperature, cloudiness, wind speed, and humidity. This site provides the source dataand visiualization created as part of the analysis, as well as explanations and descriptions of any trends and correaltions witnessed.
